# CN-DX23TT10-TranThanhTan-HeThongQuanLyCongVanDiDen
## 📖 Giới thiệu đề tài
Hệ thống Quản Lý Công Văn Đi Đến được xây dựng nhằm hỗ trợ cơ quan/doanh nghiệp trong việc:

- Quản lý công văn đến
- Quản lý công văn đi
- Theo dõi trạng thái xử lý văn bản
- Tìm kiếm và lưu trữ công văn
- Phân quyền người dùng

Hệ thống giúp tối ưu quy trình xử lý văn bản, giảm thất lạc hồ sơ và nâng cao hiệu quả quản lý.
## 🎯 Chức năng chính
### 👤 Người dùng
- Đăng nhập / đăng xuất
- Đổi mật khẩu
- Quản lý thông tin cá nhân
- ### 📥 Công văn đến
- Thêm công văn đến
- Cập nhật thông tin công văn
- Xóa công văn
- Tìm kiếm công văn
- Theo dõi trạng thái xử lý
- ### 📤 Công văn đi
- Tạo công văn đi
- Quản lý danh sách công văn
- In và lưu trữ công văn
- ### 🛠️ Quản trị hệ thống
- Quản lý tài khoản
- Phân quyền người dùng
- Thống kê số lượng công văn

- ## 🧰 Công nghệ sử dụng
- **Backend:** Spring Boot  
- **Frontend:** HTML, CSS, JavaScript, Thymeleaf  
- **Database:** MySQL  
- **IDE:** IntelliJ IDEA / VS Code  
- **Version Control:** Git & GitHub
